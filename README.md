###渐变
```css
{
  width: 200px;
  height: 200px;
  border:1px solid #333;
  background: -webkit-linear-gradient(top,#fff,#f8f8f8);
  background: -moz-linear-gradient(top,#fff,#f8f8f8);
  background: -ms-linear-gradient(top,#fff,#f8f8f8);
  background: linear-gradient(top,#fff,#f8f8f8);
}
           
```
top  从上往下
top left  从左上往右下
